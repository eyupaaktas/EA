Algoritma Analizi ve Big O Notasyonu

20 Ocak 2024

Algoritmaların performansını nasıl ölçeriz? Big O Notasyonu nedir? Bu yazıda temel algoritma analizi konularını ele alacağız.

Big O Notasyonu Nedir?

Big O Notasyonu, bir algoritmanın çalışma süresinin üst sınırını gösteren matematiksel bir gösterimdir.

Yaygın Big O Notasyonları

  • O(1) - Sabit Zaman
  • O(log n) - Logaritmik Zaman
  • O(n) - Doğrusal Zaman
  • O(n²) - Karesel Zaman

Örnek Analiz

// Basit bir lineer arama algoritması
for(int i = 0; i < n; i++) {
    if(array[i] == target)
        return i;
}
// Zaman Karmaşıklığı: O(n)
